# ScieZone Intellecta

This project is a simple static website made with HTML, CSS, and JavaScript.

## Project files

- HTML pages for the different sections of the site
- CSS styles in style.css
- JavaScript functionality in script.js

## How to view locally

Open any HTML file in a browser, for example:

- ScieZoneIntellectaDefault.html

## Development notes

- This is a front-end static site.
- No build step is required.
- You can edit the HTML, CSS, and JavaScript files directly.
